## 第1级：图片智能裁剪
目标用时：1天

参照小程序：仿照“云开发智能图像”小程序中的”人脸检测与分析“分类下的”智能裁剪“

框架：原生微信小程序、web（如Reat、Vue）、框架（Taro、Uniapp）选其一，初学者建议用原生微信小程序来实现

### 功能列表（题目要求）：

* 选择图片
* 上传到云存储
* 在云函数中，使用[云开发扩展能力](https://console.cloud.tencent.com/tcb/add)
  * 图像安全审核
  * 图像智能裁剪
* 在小程序侧显示结果
